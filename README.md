## Development
```
# First, clone the project:
git clone https://github.com/atomery/translatium.git
cd translatium

# set API keys
export REACT_APP_OCR_SPACE_API_KEY=...

# install the dependencies
yarn

# Run development mode of Translatium
yarn electron-dev

# Build for production
yarn dist
```
