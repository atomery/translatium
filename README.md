## Development Guide
### macOS
**Requirements**
1. Xcode (latest).
2. Node.js 8+.

**Development**
`yarn electron-dev`

**Release**
`yarn dist-electron`

---

### Windows 10
**Requirements**
1. Visual Studio 2017.
2. Node.js 8+.

**Development**
1. `yarn build`
2. Open `windows.sln` with Visual Studio and run the project.

**Release**
1. `yarn build`
2. Open `windows.sln` with Visual Studio and build the project.
